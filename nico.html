FLEX-BOX

Diseño de Layout


¿Que es?

Es un diseño unidimensional en CSS que permite distribuir el espacio entre los elementos de un contenedor y alinear sus contenidos de manera eficiente.

Estructura Basica

Para utilizar Flexbox, necesitas display: flex en el contenedor padre. Los elementos dentro del contenedor se alinearán en una fila de manera predeterminada.
